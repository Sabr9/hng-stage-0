# HNG Stage 0 - Todo Card

A simple, dynamic Todo Card built with HTML, CSS, and vanillina JavaScript. 

## Features
- **Dynamic Countdown**: Calculates the days remaining until the deadline using JavaScript.
- **Responsive Design**: Styled with a professional "Wheat and Chocolate" color palette.
- **HNG Requirements**: All elements include the required `data-testid` attributes.

## Tech Stack
- HTML5
- CSS3 (Flexbox)
- JavaScript (ES6+)
