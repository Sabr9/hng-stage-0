# HNG Stage 1A - Dynamic Todo Card
A professional, interactive Todo Card built with a focus on state management and accessibility. This project demonstrates the ability to handle DOM manipulation, real-time date calculations, and responsive styling.

# Live Demo
stage-0-hng.netlify.app

- Features
Dynamic Status Toggle: Real-time "Pending" to "Done" status updates with visual strike-through feedback.

Time Engine: Calculates and displays time remaining or "Overdue" status based on user input.

Dual Mode Interface: Seamlessly toggle between "View" and "Edit" modes.

Collapsible Details: Interactive "Show More/Less" functionality for task descriptions.

HNG Compliance: Fully instrumented with data-testid attributes for automated grading.
